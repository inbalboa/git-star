# Git star

Type 'git star' to show the GitHub stars and forks count.

## Usage
```bash
git star
````

## Examples
```bash
$ git star
https://github.com/inbalboa/git-star
★ Star 13 · ⑂ Fork 7
```

## Installation
Just download 'git-star' your preferred way and add the script somewhere into your path (e.g. /usr/local/bin).

## Requirements
* jq
